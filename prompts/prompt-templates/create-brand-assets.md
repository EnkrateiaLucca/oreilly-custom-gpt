# Create Brand Assets

## Source
https://claude.com/resources/use-case/Create-brand-assets

## Template Prompt

Design {ASSET_TYPES|business cards,flyers,social media graphics,posters,brochures} for {BUSINESS_NAME}. I need:

**Asset Specifications:**
- {ASSET_1|Business cards} ({DIMENSIONS_1|3.5" × 2",standard size,custom dimensions})
- {ASSET_2|Flyer,Poster,Banner} ({DIMENSIONS_2|18" × 24",11" × 17",custom size})
- {ADDITIONAL_ASSETS|Letterhead,Email signature,Social media templates}

**Branding Requirements:**
I've uploaded brand guidelines that include:
- Visual identity: {AESTHETIC|Watercolor-inspired,Modern minimal,Vintage,Bold and graphic}
- Color palette: {COLOR_DESCRIPTION|Dawn-inspired hues,Brand-specific colors,Neutral tones}
- Design elements: {DESIGN_ELEMENTS|Watercolor circles,Geometric shapes,Organic textures,Typography-focused}
- Texture/patterns: {TEXTURES|Paper grain texture,Smooth gradient,Distressed overlay,Clean flat design}

**Content to Include:**
- {BUSINESS_INFO|Contact information,Tagline,Logo,Website}
- {INTERACTIVE_ELEMENTS|QR code with rounded corners,Barcode,Social media icons}
- {PROMOTIONAL_CONTENT|Special offer,Event details,Product showcase,Services list}

**Technical Print Specifications:**
- Cardstock: {CARDSTOCK|32pt,16pt,110lb cover,Custom weight}
- Finish: {FINISH|Soft-touch coating,Matte,Gloss,Uncoated}
- Include {PRINT_ELEMENTS|bleeds,crop marks,color bars,registration marks}
- {COLOR_MODE|CMYK,RGB,Pantone} color mode
- Resolution: {RESOLUTION|300 DPI,600 DPI,print-ready quality}

**Design Direction:**
- Style: {STYLE|Professional and sophisticated,Playful and creative,Corporate and clean,Artistic and unique}
- Typography: {TYPOGRAPHY|Modern sans-serif,Classic serif,Handwritten,Mixed hierarchy}
- Layout: {LAYOUT|Information-dense,Spacious and minimal,Asymmetric,Grid-based}

Please ensure all designs maintain consistent branding across formats and are ready for professional printing.

## Use Case Description

This template helps generate professional marketing materials by processing brand guidelines and translating visual identity requirements into print-ready design files. The workflow shows how to create customized branded assets (business cards, flyers, posters, etc.) that maintain consistent branding while meeting technical printing specifications.

## Variables Explanation

- **ASSET_TYPES**: Types of marketing materials to create
- **BUSINESS_NAME**: Name of the business or organization
- **ASSET_1-2**: Specific assets needed
- **DIMENSIONS_1-2**: Exact size specifications for each asset
- **ADDITIONAL_ASSETS**: Other materials that might be needed
- **AESTHETIC**: Overall visual style of the brand
- **COLOR_DESCRIPTION**: Color palette approach
- **DESIGN_ELEMENTS**: Specific visual elements that define the brand
- **TEXTURES**: Surface quality or patterns to incorporate
- **BUSINESS_INFO**: Essential information to include
- **INTERACTIVE_ELEMENTS**: Functional elements like QR codes
- **PROMOTIONAL_CONTENT**: Marketing messages or information
- **CARDSTOCK**: Paper weight and quality
- **FINISH**: Surface treatment of printed materials
- **PRINT_ELEMENTS**: Technical printing requirements
- **COLOR_MODE**: Color space for printing
- **RESOLUTION**: Image quality specification
- **STYLE**: Design approach and mood
- **TYPOGRAPHY**: Font style and hierarchy
- **LAYOUT**: Spatial organization approach

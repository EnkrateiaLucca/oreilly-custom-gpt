# Create a Custom Webpage

## Source
https://claude.com/resources/use-case/create-a-custom-webpage

## Template Prompt

Help me create a professional {PAGE_TYPE|portfolio,landing page,personal website,project showcase,resume site} HTML page to {PURPOSE|showcase my professional career and accomplishments,promote my services,present my research,display my creative work}.

I've uploaded:
- {DOCUMENT_1|Resume,CV,Project descriptions}
- {DOCUMENT_2|Design inspiration files,Brand guidelines,Sample layouts}
- {DOCUMENT_3|Images,Content documents,Additional materials}

**Design Requirements:**
- Style: {AESTHETIC|Modern and minimal,Bold and creative,Professional and clean,Artistic and unique}
- Color scheme: {COLORS|Based on uploaded inspiration,Specific colors provided,Neutral and professional,Vibrant and energetic}
- Layout: {LAYOUT_STYLE|Single-page scroll,Multi-section,Grid-based,Asymmetric}
- Avoid {AVOID|generic templates,overly complex designs,excessive animations,cluttered layouts}

**Content Sections:**
- {SECTION_1|Professional summary,Hero section,Introduction}
- {SECTION_2|Work experience,Projects,Portfolio pieces}
- {SECTION_3|Skills,Services,Research areas}
- {SECTION_4|Contact information,Call-to-action,Social links}

**Technical Requirements:**
- {RESPONSIVE|Fully responsive,Mobile-first,Desktop-optimized} design
- {FEATURES|Smooth scrolling,Interactive elements,Animations,Static content}
- Ready to deploy via {HOSTING|Netlify,Vercel,GitHub Pages,Custom hosting}

## Use Case Description

## Variables Explanation

- **PAGE_TYPE**: Type of website being created
- **PURPOSE**: Main goal of the website
- **DOCUMENT_1-3**: Types of files being uploaded
- **AESTHETIC**: Overall design style
- **COLORS**: Color scheme approach
- **LAYOUT_STYLE**: Page layout structure
- **AVOID**: Design elements to exclude
- **SECTION_1-4**: Main sections of the website
- **RESPONSIVE**: Responsiveness priority
- **FEATURES**: Interactive features to include
- **HOSTING**: Intended hosting platform
